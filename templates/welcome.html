Welcome to the GeoFusion API

This API allows you to join geospatial data with tabular data. Use the following endpoints to interact with the API:

1. Join Data Endpoint: /tjs/api/joindata

Description: Fetches GeoJSON data from a GeoServer and joins it with a CSV file based on specified keys.

Parameters:

  • FrameworkURI: The URL to the GeoServer providing the GeoJSON data.
  • layer: The name of the layer in the GeoServer.
  • GetDataURL: The URL to the CSV file containing the tabular data.
  • FrameworkKey: The key in the GeoJSON data to join on.
  • AttributeKey: The key in the CSV data to join on.

Example:

GET /tjs/api/joindata?FrameworkURI=http://localhost:8081/geoserver/topp/ows&layer=topp:states&GetDataURL=https://example.com/data.csv&FrameworkKey=STATE_NAME&AttributeKey=StateName

2. Get GeoJSON Tile Endpoint: /tjs/api/tms/

Description: Fetches GeoJSON tiles from a GeoServer.

Parameters:

  • FrameworkURI: The URL to the GeoServer providing the GeoJSON tiles.
  • GetDataURL: The URL to the CSV file.
  • FrameworkKey: The key in the GeoJSON data.
  • AttributeKey: The key in the CSV data.

Example:

GET /tjs/api/tms/?FrameworkURI=http://localhost:8081/geoserver/topp/ows&GetDataURL=https://example.com/data.csv&FrameworkKey=STATE_NAME&AttributeKey=StateName
